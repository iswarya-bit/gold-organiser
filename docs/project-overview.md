# Personal Gold Loan Organizer

## Objective

A secure application that helps users manage their gold loans in one place.

## Features

- User Registration
- User Login
- Gold Loan Management
- Smart Reminders
- OCR Extraction
- Security Verification

## Tech Stack

Frontend:
- React

Backend:
- Spring Boot

Database:
- MySQL

Security:
- Spring Security
- JWT

Future:
- OCR
- AI Extraction
